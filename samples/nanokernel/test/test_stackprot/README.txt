Title: test_stackprot

Description:

This test verifies that stack canaries operate as expected in the nanokernel.

---------------------------------------------------------------------------

This nanokernel project outputs to the console.  It can be built and executed
on QEMU as follows:

    make pristine
    make nanokernel.qemu
