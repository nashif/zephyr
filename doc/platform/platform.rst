======================
Platform Documentation
======================

Zephyr supports the following hardware platforms and configurations:

+----------------------------------------+----------------+--------------+
|                                        | Platform       |              |
| Hardware Platform                      | Configuration  | Architecture |
|                                        | Name           |              |
+========================================+================+==============+
| Freescale Freedom Development Platform | fsl_frdm_k64f  | ARM v7E-M    |
+----------------------------------------+----------------+--------------+
| Galileo                                | quark          | X86          |
| Galileo (Gen 2)                        |                |              |
+----------------------------------------+----------------+--------------+

.. note::
       Customers can create custom platform configurations for any Zephyr-supported hardware
       platform.
